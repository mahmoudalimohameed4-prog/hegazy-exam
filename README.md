# منصة الاختبارات الإلكترونية (Quiz Platform)

نظام متكامل لإدارة وأداء الاختبارات الإلكترونية بتصميم عصري ومتجاوب.

## المميزات
- **نظام دخول ثنائي:** عبر رقم الهاتف (OTP) أو الرقم القومي.
- **لوحة تحكم المعلم:** إنشاء، حذف، ومتابعة نتائج الطلاب.
- **واجهة الطالب:** أداء الاختبارات مع مؤقت زمني وحساب فوري للدرجات.
- **تصميم مميز:** استخدام Tailwind CSS مع دعم كامل للغة العربية (RTL).

## طريقة التشغيل

### المتطلبات
- Node.js مثبت على جهازك.
- قاعدة بيانات MongoDB تعمل (محلياً على `mongodb://localhost:27017/quiz_platform`).

### تشغيل الخلفية (Backend)
1. افتح مجلد `backend`.
2. قم بتشغيل الأمر: `npm install`.
3. قم بتهيئة حساب المعلم الافتراضي: `npm run seed`.
4. ابدأ الخادم: `npm run dev`.
   - يعمل الخادم على المنفذ `5000`.

### تشغيل الواجهة (Frontend)
1. افتح مجلد `frontend`.
2. قم بتشغيل الأمر: `npm install`.
3. ابدأ الواجهة: `npm run dev`.
   - تعمل الواجهة على المنفذ `5173`.
